Detailed Description

봇프로그램의 기본적인 뼈대 구조를 정의한 class 로서, 스타크래프트 경기 도중 발생하는 이벤트들을 GameCommander class 인스턴스에게 전달합니다. MyBotModule class는 수정을 하지 말고, 실제 봇프로그램 개발은 GameCommander class 를 수정하는 형태로 진행하도록 합니다
